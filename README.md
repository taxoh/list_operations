# list_operations

Скрипт с веб-интерфейсом для работы со строковыми списками.

Возможности:

- умножение списков (mul)

- вычитание (diff)

- пересечение (intersect)

- мультиплексирование (mux)

- демультиплексирование (demux)

- перемешивание (shuffle)

- удаление дублей (unique)

- соединение (merge)
